3.2 Attribute Analysis

对于问题“单词的属性是否会影响结果百分比”，我们的回答是“是”。

4 Task 2: Prediction of Result Percentages

利用单词的特征对 777 项结果百分比做直接估计是较为困难的。在 4.1 部分中，基于结果百分比本身，我们首先将所有测试过的单词分为了若干类，满足每一类中的所有单词的结果百分比尽量接近。随后，我们在 4.2 部分中利用深度学习构建出了一个以单词特征作为输入，分类编号作为输出的预测模型，并以单词 eerie\tt eerieeerie 为例说明了如何通过分类编号得到具体的结果百分比预测。在最后的 4.3 部分，我们对预测的信心与不确定因素做了分析。

4.1 Classification

首先确定分类数 KKK。KKK 不应过小，否则每一类中单词的结果百分比将不会呈现出较为明显的共同特征；也不应过大，否则对深度学习过程不利。此处取 K=15K = 15K=15。

我们使用 KKK-means 聚类算法完成这一过程。对于每一个单词，设其结果百分比分别为 p1,⋯ ,p6,pXp_1, \cdots, p_6, p_{\rm X}p1​,⋯,p6​,pX​，它们构成了 777 维坐标向量 p=(p1,⋯ ,p6,pX)\boldsymbol p = (p_1, \cdots, p_6, p_{\rm X})p=(p1​,⋯,p6​,pX​)。对 777 维空间中的任意两个点 P,QP, QP,Q，定义它们的欧几里得距离为聚类算法的距离函数，即设两点坐标向量为 p=(p1,⋯ ,p6,pX),q=(q1,⋯ ,q6,qX)\boldsymbol p = (p_1, \cdots, p_6, p_{\rm X}), \boldsymbol q = (q_1, \cdots, q_6, q_{\rm X})p=(p1​,⋯,p6​,pX​),q=(q1​,⋯,q6​,qX​)，则它们之间的距离

dist(P,Q)=(pX−qX)2+∑i=16(pi−qi)2.{\rm dist}(P, Q) = \sqrt{(p_{\rm X} - q_{\rm X})^2 + \sum_{i=1}^6 (p_i - q_i)^2}.dist(P,Q)=(pX​−qX​)2+i=1∑6​(pi​−qi​)2​.

选取该函数意味着被归为同一类的点，它们的各项结果百分比差距不会过大，这与我们的分类期望是相符合的。

在成功将所有单词分为 K=15K = 15K=15 类后，我们通过两个典型的统计数据：平均值和方差，来展示我们的分类效果。可以看到，平均值和方差接近的单词被分类在了同一个聚类中，因此对于未来的单词，如果我们顺利地确定了它所在的类别，那么预测出它的各项结果百分比也就较为容易了。

4.3 Model Analysis

模型的不确定性主要来自于两个方面，它们与上述分类与深度学习两个步骤分别对应。

在分类上，由于所有单词的结果百分比分布没有明确界限，因此处于不同分类中的单词结果差距并非一定明显，两个距离极小的点被分到了不同的类中的情况完全可能出现。同时，KKK-means 算法在一定程度上基于随机化，这也引入了不确定因素，例如，类中可能包含这样的点，它与类中其他点相距很远，这会给该类对应的结果预测带来一定的麻烦。

在深度学习上，不确定性也会由随机的初始参数引入。有时，一个不当的初始参数容易使学习陷入局部最优结果（而非全局最优结果），这意味着需要通过多次学习来寻找较为良好的参数。同时，略大于 60%60\%60% 的已有单词预测准确率尽管理论上已经不错，但假设你对深度学习了解甚少，这一数值似乎并不会使你信服。

不过，尽管模型具有不确定性，我们对于单词 eerie\tt eerieeerie 的预测却较有信心。一方面，在连续的 555 次学习中，神经网络有 333 次对于单词 eerie\tt eerieeerie 给出的预测类别编号都是 222，这表明对于单词 eerie\tt eerieeerie，从特征向量到分类编号的映射较为明显和稳定；另一方面，这一结果在经验上也较为正确——eerie\tt eerieeerie 通常不会作为一个玩家在首次猜测中使用的单词，因此预测的 p1=0p_1 = 0p1​=0，同时，该单词在使用频率中处于中间水平，且含有三个元音字母 e\tt ee 的特征较为特殊，因此认为最后才猜中的概率 p6=21%p_6 = 21\%p6​=21% 以及猜不出的概率 pX=5%p_{\rm X} = 5\%pX​=5% 也是合理的。
